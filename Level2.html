Village
      Has a certain number of houses
      Has a certain number of pigs
      Has a certain number of wolves
      Has a certain number of restaurants
A typical Village would have
      Number of houses = 28
      Number of pigs = 24
      Number of wolves = 33

Pig’s Revenge on the Wolves
      Village Vaughan is a new Village
           Number of houses = 32
           Number of pigs = 24
           Number of wolves = 33

Mammals
      Have a certain number of legs
      Have a certain number of brains
      Have a certain number of eyes
      Have a certain house number
      Have a gender
      Have a species
      Either have fur or not
A standard mammal would start with
      Four legs
      One brain
      Two eyes
When another mammal asks for your name
      You respond with your name

Pig extends the idea of mammal.
      A pig
           Has a monocle
           Has a hat
           Has a level of agreeableness
           Has a level of hunger
           Has a certain number of houses he’s blown down
      For a given pig
           He will start at a level of max hunger.
           He will start the day without having blown any houses down.

      setHungerLevel
           hungerLevel gets reduced by 20%

      getHungerLevel
           Tell the mammal who’s asking how hungry you are

      blowDownWolfHouse (Mammal wolf)
           Add one to the number of wolf houses this pig has blown down
           Print “Oh my gosh! {the specified house owned by the specified wolf} has been blown down.”

      Pigs extends the idea of Mammals.
           Every pig has a monocle.
           Every pig has a hat.
           Every pig has a level of hunger.
           Every pig has a level of agreeableness.
           Every pig will blow down a certain amount of houses.
      For a given pig,
           He will start at a level of max hunger.
           He will start the day without having blown any houses down.
      setHungerLevel
           Hunger level decreases by 20%.
      getHungerLevel
           Tell them how hungry you are.
      blowDownWolfHouse (name, houseNumber)
           Add one to the number of wolf houses this pig has blown down
           Print “Oh my gosh! {the specified house owned by the specified wolf} has been blown down."
